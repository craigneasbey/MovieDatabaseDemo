A demonstration of ASP.NET Core and Angular

https://github.com/craigneasbey/MovieDatabaseDemo

## Features
* Add people: actors, producers, script writers, set designers, etc
* Add movies
* Search for a move or person

* .NET 6 SDK
* nodejs 16
